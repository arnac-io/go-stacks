# Bns Get Name Price Response

Fetch price for name.

## Structure

`BnsGetNamePriceResponse`

## Fields

| Name     | Type     | Tags     | Description                            |
| -------- | -------- | -------- | -------------------------------------- |
| `Units`  | `string` | Required | -                                      |
| `Amount` | `string` | Required | **Constraints**: _Pattern_: `^[0-9]+$` |
